h1. Vines

Swing around your code.

h1. How To Install

<pre>
  mkdir -p ~/Library/Application\ Support/TextMate/Bundles/
  cd ~/Library/Application\ Support/TextMate/Bundles/
  git clone git://github.com/timcharper/vines.tmbundle.git Vines.tmbundle
</pre>


h2. Features

* Select Previous Parenthesis   
* Select Next Parenthesis       
* Find Next String/Symbol       
* Find Previous String/Symbol   
* Select Next Bracket/Brace     
* Select Previous Bracket/Brace 
* Select Tag
